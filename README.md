# ParfumPasCher

## Web Content Management Assignment

This project was developed as part of my **Web Content Management** subject assignment. It is a basic web based project created for academic purposes and demonstrates concepts related to website design, content organization, and web development.

### Project Description
ParfumPasCher is a website project focused on presenting perfume related content through a structured and user friendly interface. The project aims to showcase essential web development concepts such as layout design, navigation, and content presentation.

### Technologies Used
- HTML
- CSS
- JavaScript

### Purpose
The purpose of this assignment is to understand and apply the fundamentals of Web Content Management, including:

- Website structure and organization
- User interface design
- Content presentation
- Basic frontend development concepts
